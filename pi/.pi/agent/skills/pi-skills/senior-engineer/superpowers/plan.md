# Superpowers – Planning Module

> **Activation condition:** Only follow this when the user has opted into the senior engineer workflow (directly or via the root Senior Engineer skill).

Template checklist:

1. **Goals & scope** – copy the clarified requirements into `task_plan.md`.
2. **Phases / tasks** – break the work into ordered steps with checkboxes.
3. **Test strategy** – enumerate unit/integration/manual tests you expect to run.
4. **Resources** – list files, APIs, or docs you’ll need.
5. **Definition of done** – state exactly how you’ll know the task is complete (tests passing, docs updated, etc.).

Update `progress.md` once the plan is in place and confirm with the user before executing.
