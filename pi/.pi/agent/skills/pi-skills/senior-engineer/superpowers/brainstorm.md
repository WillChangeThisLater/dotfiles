# Superpowers – Brainstorm Module

> **Use only when explicitly requested by the user or when `senior-engineer/SKILL.md` tells you to activate Superpowers.**

1. Restate the problem in your own words.
2. List at least 2–3 possible approaches (label Pros / Cons).
3. Identify unknowns, external dependencies, or blockers.
4. Decide on the primary approach and capture the rationale in your planning file (e.g., `task_plan.md`).
5. Note any fallback strategies if the primary plan fails.
