# Superpowers – Review & Wrap-Up Module

> **Activation condition:** Only apply this section when operating under the senior engineer / Superpowers workflow.

1. **Checklist pass** – re-read the plan and verify every checkbox/phase is complete. Update statuses if needed.
2. **Tests & evidence** – rerun the full suite or manual steps; include output snippets or screenshot paths.
3. **Documentation** – summarize changes in `progress.md`, note decisions in `findings.md`, and mention follow-up work if any.
4. **User hand-off** – report results clearly, referencing files, commits, and test evidence.
5. **Exit Superpowers mode** – tell the user you’re leaving the structured workflow (unless they want to keep it on for future tasks).
